# 🔐 RSA Encryption GUI (Python)

## 📌 Overview
This project demonstrates RSA encryption using Number Theory concepts in Information Security. It includes a GUI built with Tkinter.

## 🚀 Features
- Encrypt numeric message
- Decrypt encrypted message
- Displays public and private keys

## 🧠 Concepts Used
- Prime Numbers
- Modular Arithmetic
- Euler’s Totient Function
- Modular Inverse

## 🖥️ Technologies Used
- Python
- Tkinter (GUI)

## ▶️ How to Run
```bash
python main.py
